Методи за работа в Selenium:

I. Основни методи:

1. get();
2. getText();
3. click();
4. submit();
5. sendKeys();
6. clear();
7. quit();
8. Select();
9. Try Catch
10. Action class
11. Hover
12. Drag and Drop


Примери:


1. get(); - методът се използва за извикване на определен сайт:
Пример:
driver.get("https://www.youtube.com");. Изписана по този начин, командата ще
извика сайт www.youtube.com, като ще впише адреса на сайта в адрес бара на браузъра,
който автоматично ще го достъпи.

2. getText(); - използва се при сличаи, в които е нужно да се копира даден
текс от интердейса и да се изпринти в конзолата на компилатора.
Подходящ е за сравнения на текст.
Пример:
Ако в края на теста бива показано писмено съобщение за успех на извършената операция,
то от този текст може да се създаде стринг, който е еднакъв със съобщението от екрана:
String 'text1' = driver.findElement(By.id("id")).getText();
String 'text2' = "Operation Successful";
Така променливата 'text1' ще копира написаното в интерфейса и ще го съхрани.
Проверката за еднаквост между създадения стринг и онзи, който е изписан в интерфейса,
се изпълнява по следния начин:
if(text1.equalsIgnoreCase(text2)){
System.out.println("Test passed!");
}
else{
System.out.println("Test failed");
}

3. click(); - методът е подходящ за манипулиране на бутони от всякакъв вид, може да бъде заместен с друг
сходен по действие такъв (например: .submit();).
Пример:
driver.findElement(By.id(id)).click();

4. submit(); - използва се при бутони от тип 'submit'. Действието му е сходно с това на метод click();
Пример:
driver.findElement(By.id(id)).submit();

5. sendKeys(); - методът се използва главно за вписване на стринг в 'input' поле.
Пример:
driver.findElement(By.id(id)).sendKeys("text");
Подходящ е още за имитиране на команди от клавиатура, като:
.sendKeys(Keys.DELETE);- ще извърши действие като метод .clear();

6. clear(); - този метод ще изпълни премахване на даден стринг от 'input' поле.
Подходящ е за изтриване на полетa, които съдържат дата (филтри за период, дата или полета със съхранен
стринг).
Пример:
driver.findElement(By.id(id)).clear();

7. quit(); - методът ще затвори браузъра след края на теста.
Пример:
driver.quit();

8. Select(); - методът се използва при селект полета, в които се съдържат различни опции.
Пример:
Select select = new Select();
select.selectByIndex(index);

9. Try Catch:
Пример:
public class Demo {
public static void main(String[ ] args) {
try {
driver.findElement(By.xpath(xpath)).click();
} catch (Exception e) {
System.out.println("Something went wrong.");
}
}
}

10. Actions class- Класът Actions е способност, предоставена от Selenium за обработка
на събития с клавиатура и мишка. Класът е подходящ е за операции като Check card в интерфейс Cashdesk.
Пример:
Actions action = new Actions(driver);
action.moveToElement(element).click().perform();

11. Hover- част от клас Actions, подходящ за зависване над даден обект или елемент от интерфейса.
Приложим е за интерфейси, в които има икони, в които се съдържат бутони, а те се визуализират
след зависяване с курсора. Такива са бутоните за стартиране на игра в 'Play' или 'Demo' режим
в сайт PalmsBet.
Пример:
WebElement funBtn = driver.findElement(By.id("fun-button"));
Actions action = new Actions(driver);
action.moveToElement(funBtn).perform();

12. Drag and Drop- това действие се извършва с помощта на мишката, когато потребителят премести
уеб елемент от една точка и го постави в друга точка на интерфейса.
При автоматизиране на това действие в web апликация e нужнo да се изпълни следното:
1. Обект, който ще бъде преместван- SourceLocator.
2. Обект или поле, в който/което ще бъде поставен SourceLocator- DestinationLocator.
Пример:
Actions action = new Actions(driver);
action.dragAndDrop(SourceLocator, DestinationLocator).build().perform();
